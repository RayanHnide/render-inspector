# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and inclusive for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Give actionable feedback.
- Focus on ideas, not individuals.

## Unacceptable Behavior

- Harassment, hate speech, or personal attacks
- Trolling or inflammatory comments
- Publishing private information without consent

## Enforcement

Project maintainers may remove comments, close threads, or restrict participation for behavior that violates this code.
